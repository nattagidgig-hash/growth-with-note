# รายละเอียดหน้าที่

## 1. Project Manager

### รับผิดชอบ

- วิสัยทัศน์โครงการ
- งบประมาณ
- การเงิน
- จัดซื้อ
- ประสานหน่วยงาน
- ผู้ปกครอง
- ความเสี่ยง
- รายงานผล

KPI

- โครงการดำเนินตามแผน
- งบไม่บานปลาย
- ทีมทำงานราบรื่น

---

## 2. Program Coordinator

### รับผิดชอบ

- Run Sheet
- ตารางเรียน
- การเปลี่ยนกิจกรรม
- การจับเวลา
- Flow ทั้งวัน
- Brief ทีม

KPI

- ไม่มีช่วงเวลาว่าง
- ทุกกิจกรรมตรงเวลา

---

## 3. Music Instructor

### รับผิดชอบ

- ออกแบบกิจกรรม
- สอน
- ดูแลเด็ก
- ประเมินผล

KPI

- เด็กเข้าใจบทเรียน
- เด็กมีส่วนร่วม
- เด็กสนุก

---

## 4. Operations & Student Support

### รับผิดชอบ

ผู้เรียน

- ลงทะเบียน
- เช็คชื่อ
- ดูแลผู้ปกครอง

อาหาร

- น้ำ
- ของว่าง
- อาหารกลางวัน

สถานที่

- โต๊ะ
- เก้าอี้
- เครื่องเสียง
- เครื่องดนตรี

อุปกรณ์

- กระดาษ
- ปากกา
- สายไฟ
- ปลั๊ก

เอกสาร

- รูปภาพ
- วิดีโอ
- แบบประเมิน

KPI

- ทุกอย่างพร้อมก่อนเริ่มเรียน
- ไม่มีอุปกรณ์ขาด
- ผู้เรียนได้รับการดูแลครบ

# Dashboard ที่ควรตรวจทุกสัปดาห์

1. จำนวนผู้มาเรียน
2. จำนวนผู้ขาด
3. งบที่ใช้จริง
4. ค่าใช้จ่ายคงเหลือ
5. ความพึงพอใจของผู้เรียน
6. ปัญหาที่พบ
7. สิ่งที่ต้องปรับปรุง
8. Action ของสัปดาห์ถัดไป

เมื่อทุกสัปดาห์จบลง Dashboard นี้จะเป็นฐานข้อมูลสำหรับพัฒนาโครงการอย่างต่อเนื่อง และช่วยให้การรายงานต่อผู้สนับสนุนหรือหน่วยงานภายนอกมีข้อมูลที่ชัดเจนและตรวจสอบได้
